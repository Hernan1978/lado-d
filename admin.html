Integrar con Google Sheets

📋 Pasos para conectar tu Sheet:

  1. Crear un nuevo Google Sheet con las siguientes columnas:
    id, title, category, preview, date, image, link
  2. Agregar tus artículos como rows en el Sheet
  3. Hacer clic en "Compartir" → "Compartir con todos"
  4. Ir a "Archivo" → "Descargar" → "CSV (.csv)"
  5. Usar el enlace CSV para cargar los datos dinámicamente
  6. Ver código de integración en app.js (última sección)

💡 Nota: El panel admin está preparado para enviar datos a tu Sheet. Cuando tengas tu Sheet listo, podemos integrar la API para que los artículos se publiquen automáticamente.
